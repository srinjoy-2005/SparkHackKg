Module: main

📁 main.py

Main entry point — orchestrates calculator operations.

Functions:

main()

Initialize calculator and run demo operations.

Steps:

  1. Create Logger with config.LOG_LEVEL
  2. Initialize Calculator with logger
  3. Initialize AdvancedOps with calculator
  4. Run basic operations (add, subtract)
  5. Run advanced operations (sqrt_safe, power_safe)
  6. Log completion
