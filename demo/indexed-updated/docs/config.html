Class: Config

📁 config.py

Global configuration settings.

Class Attributes:

LOG_LEVEL = "INFO"

Default logging level for the application.

MAX_PRECISION = 10

Maximum precision for floating-point operations.

TIMEOUT_SECONDS = 30

Default timeout for operations (in seconds).

Graph Relationships:

  • IMPORTED BY: logger.Logger
  • IMPORTED BY: main.main()

Usage Example:

from config import Config
from logger import Logger

# Access global settings
print(Config.LOG_LEVEL)          # "INFO"
print(Config.TIMEOUT_SECONDS)    # 30

# Use in logger
logger = Logger(Config.LOG_LEVEL)

Purpose:

This module centralizes all configuration constants, making it easy to adjust application behavior without modifying individual modules. It's the single source of truth for application settings.
