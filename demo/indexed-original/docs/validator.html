Module: validator

📁 validator.py

Input validation utilities for numeric operations.

Functions:

validate_number(value)

Ensure value is a valid number for operations.

Checks:

  • Type must be int or float
  • Must not be NaN (Not a Number)
  • Must not be Inf (Infinity)

Raises:

  • TypeError if value is not a number
  • ValueError if value is NaN or Inf

Graph Relationships:

  • CALLED BY: calculator.Calculator.add()
  • CALLED BY: calculator.Calculator.subtract()
  • CALLED BY: calculator.Calculator.multiply()
  • CALLED BY: operations.AdvancedOps methods

Usage Example:

from validator import validate_number

# Valid
validate_number(42)        # OK
validate_number(3.14)      # OK

# Invalid - raises exceptions
validate_number("42")      # TypeError
validate_number(float('nan'))   # ValueError
validate_number(float('inf'))   # ValueError
