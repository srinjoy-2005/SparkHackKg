Class: Logger

📁 logger.py

Simple logger for tracking operations.

Constructor:

Logger(level: str = "INFO")

Initialize logger with given level.

Parameters: level ∈ {INFO, DEBUG, ERROR}

Methods:

info(message: str)

Log info-level message if level is INFO or DEBUG.

debug(message: str)

Log debug-level message if level is DEBUG.

error(message: str)

Log error message (always output to stderr).

Graph Relationships:

  • IMPORTS: config.Config
  • CALLED BY: calculator.Calculator methods
  • CALLED BY: main.main()

Usage Example:

from logger import Logger

logger = Logger("INFO")
logger.info("Operation started")      # Prints to stdout
logger.debug("Debug info")              # Not printed (level=INFO)
logger.error("An error occurred")      # Prints to stderr

Output Format:

[INFO] message
[DEBUG] message
[ERROR] message
