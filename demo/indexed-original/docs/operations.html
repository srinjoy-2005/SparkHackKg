Class: AdvancedOps

📁 operations.py

Wrapper for advanced operations combining Calculator with math module.

Methods:

sqrt_safe(x: float) → float

Calculate square root safely with negative number checking.

Raises: ValueError if x is negative

power_safe(base: float, exp: float) → float

Calculate power (base^exp) safely with full validation.

Dependencies: math.pow(), validator.validate_number()

divide_safe(a: float, b: float) → float

Divide with zero-check.

Raises: ValueError if b is zero

Graph Relationships:

  • USES: calculator.Calculator
  • IMPORTS: validator.validate_number
  • IMPORTS: math module
  • CALLED BY: main.main()

Usage Example:

from calculator import Calculator
from operations import AdvancedOps
from logger import Logger

logger = Logger("INFO")
calc = Calculator(logger)
adv = AdvancedOps(calc)

sqrt_result = adv.sqrt_safe(16)      # Returns 4.0
pow_result = adv.power_safe(2, 3)    # Returns 8.0
